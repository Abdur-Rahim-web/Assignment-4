## Answers to Questions

### 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
getElementById = it always returns one element.
getElementsByClassName = I can get multiple elements using a class name
querySelector = I can get first element that matches a CSS selector
querySelectorAll = I can get all elements that match a CSS selector
### 2. How do you create and insert a new element into the DOM?
By using document.createElement()
### 3. What is Event Bubbling? And how does it work?
It target an element and then moves up to its parent elements.
### 4. What is Event Delegation in JavaScript? Why is it useful?
It means adding an event listener to a parent element instead of multiple child elements.
